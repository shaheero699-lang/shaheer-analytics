# Python Layer

## Structure

| Folder | Contents |
|---|---|

- pandas, numpy — data manipulation

- scikit-learn — machine learning models
- jupyter notebooks — analysis environment

## Key Notebooks
- `customer_lifetime_value_model.ipynb` — LTV prediction model
- `revenue_forecasting.ipynb` — MRR forecast with scenario analysis
- `customer_behavior_analysis.ipynb` — Segmentation and pattern analysis
