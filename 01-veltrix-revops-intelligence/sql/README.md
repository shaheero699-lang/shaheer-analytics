# SQL Layer

## Structure

| Folder | Purpose |
|---|---|
| schema/ | Table definitions and relationships |
| staging/ | Data cleaning and transformation scripts |
| analysis/ | Business analysis queries |

## Key Analysis Queries

- `customer_retention_analysis.sql` — Cohort retention calculation
- `customer_segmentation_analysis.sql` — RFM and health segmentation  
- `revenue_performance_analysis.sql` — MRR, ARR, churn metrics
- `forecasting_analysis.sql` — Revenue forecasting logic

## Sample Query Output
Churn rate by health segment, onboarding bucket analysis,
SLA breach correlation, feature adoption by plan tier.
